**全国计算机等级考试二级Python语言程序设计考试大纲（```2018年版```）**

```白月黑羽将针对国家计算机Python语言等级考试提高专门的学习板块和题库来帮助大家，敬请关注```

## 基本要求

1. 掌握Python语言的基本语法规则。<br>
2. 掌握不少于2个基本的Python标准库。<br>
3. 掌握不少于2个Python第三方库，掌握获取并安装第三方库的方法。<br>
4. 能够阅读和分析Python程序。<br>
5. 熟练使用IDLE开发环境，能够将脚本程序转变为可执行程序。<br>
6. 了解Python计算生态在以下方面（不限于）的主要第三方库名称：网络爬虫、数据分析、数据可视化、机器学习、web开发等。<br>

## 考试内容

### Python语言基本语法元素

1. 程序的基本语法元素：程序的格式框架、缩进、[注释](http://www.python3.vip/doc/tutorial/python/0003/#注释)、[变量](http://www.python3.vip/doc/tutorial/python/0003/#变量的概念)、[命名](http://www.python3.vip/doc/tutorial/python/0003/#变量的命名规则)、[保留字](http://www.python3.vip/doc/tutorial/python/level2/1006/)、[数据类型](http://www.python3.vip/doc/tutorial/python/0002/#对象类型)、赋值语句、[引用](http://www.python3.vip/doc/tutorial/python/0003/#变量值的变化)。<br>
2. 基本输入输出函数：[input()](http://www.python3.vip/doc/tutorial/python/0006/) 、eval()、print()。<br>
3. 源程序的书写风格。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [编码规则](http://www.python3.vip/doc/tutorial/python/0001/#一些规则)<br>
4. Python语言的特点。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [为什么学Python](http://www.python3.vip/doc/tutorial/python/0001/#为什么学python)<br>

### 基本数据类型

1. 数字类型：整数类型、浮点数类型和复数类型。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [数字对象](http://www.python3.vip/doc/tutorial/python/0002/#数字对象)<br>
2. 数字类型的运算：数值运算操作符、数值运算函数。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [数字运算](http://www.python3.vip/doc/tutorial/python/0002/#数学运算)<br>
3. 字符串类型及格式化：索引、切片、基本的format()格式化方法。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [字符串](http://www.python3.vip/doc/tutorial/python/0004/)<br>
4. 字符串类型的操作：字符串操作符、处理函数和处理方法。<br>
5. [类型判断](http://www.python3.vip/doc/tutorial/python/0002/)和类型间转换。<br>

### 程序的控制结构

1. 程序的三种控制结构。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [循环](http://www.python3.vip/doc/tutorial/python/0011/)<br>
2. 程序的分支结构：单分支结构、二分支结构、多分支结构。<br>
3. 程序的循环结构：遍历循环、无限循环、[break](http://www.python3.vip/doc/tutorial/python/0011/#break-终止循环)和[continue](http://www.python3.vip/doc/tutorial/python/0011/#continue)循环控制。<br>
4. 程序的异常处理：try-except。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [异常](http://www.python3.vip/doc/tutorial/python/0019/)<br>

### 函数和代码复用

1. 函数的定义和使用。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [函数](http://www.python3.vip/doc/tutorial/python/0005/)<br>
2. 函数的参数传递：可选参数传递、参数名称传递、函数的返回值。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [函数参数](http://www.python3.vip/doc/tutorial/python/0005/#函数的参数)及[函数的返回值](http://www.python3.vip/doc/tutorial/python/0005/#函数的返回值)<br>
3. 变量的作用域：局部变量和全局变量。<br>

### 组合数据类型

1. 组合数据类型的基本概念。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [列表](http://www.python3.vip/doc/tutorial/python/0007/)、[字典](http://www.python3.vip/doc/tutorial/python/0017/)<br>
2. 列表类型：定义、索引、切片。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [列表操作](http://www.python3.vip/doc/tutorial/python/0007/#列表的sequence操作)<br>
3. 列表类型的操作：列表的操作函数、列表的操作方法。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [列表的方法](http://www.python3.vip/doc/tutorial/python/0009/#列表的方法)<br>
4. 字典类型：定义、索引。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [字典的定义](http://www.python3.vip/doc/tutorial/python/0017/#字典是什么)<br>
5. 字典类型的操作：字典的操作函数、字典的操作方法。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [字典的方法](http://www.python3.vip/doc/tutorial/python/0017/#其他常用方法)<br>

### 文件和数据格式化

1. 文件的使用：文件打开、读写和关闭。
2. 数据组织的维度：一维数据和二维数据<br>
3. 一维数据的处理：表示、存储和处理。<br>
4. 二维数据的处理：表示、存储和处理。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [文件操作](http://www.python3.vip/doc/tutorial/python/0013/)<br>
5. 采用csv格式对一二维数据文件的读写。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [excel读写](http://www.python3.vip/doc/tutorial/python/excel/)<br>

### Python计算生态

1. 标准库：[turtle 库](https://docs.python.org/3.3/library/turtle.html#module-turtle)（```必选```）、[random 库](https://docs.python.org/3/library/random.html)（```必选```） 、[time 库](http://www.python3.vip/doc/faq/python/home/#日期和时间)（可选）。<br>
2. 基本的Python内置函数。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [内置函数和库](http://www.python3.vip/doc/tutorial/python/level2/1006/)<br>
3. 第三方库的获取和安装。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [模块与库](http://www.python3.vip/doc/tutorial/python/0014/)<br>
4. 脚本程序转变为可执行程序的第三方库：Pyinstaller库（```必选```）。<br>参考[白月黑羽Python3教程](http://www.python3.vip/doc/tutorial/python/0001/) - [制作可执行文件](http://www.python3.vip/doc/tutorial/python/toexe/)<br>
5. 第三方库：[jieba库](https://pypi.org/project/jieba/)（```必选```）、[wordcloud](https://pypi.org/project/wordcloud/)（可选）。<br>
6. 更广泛的Python计算生态，只要求了解第三方库的名称，不限于以下领域：网络爬虫、数据分析、文本处理、数据可视化、用户图形界面、机器学习、web开发、游戏开发等。<br>

## 考试方式

上机考试，考试时长 120 分钟，满分 100 分。

1. 题型及分值
 + 单项选择题 40 分（含公共基础知识部分 10 分）。<br>
 + 操作题 60 分（包括基本编程题和综合编程题）。<br>
2. 考试环境
Windows 7 操作系统，建议Python3.4.2 至Python3.5.3版本，IDLE开发环境。
3. 考试时间

目前《全国计算机等级考试考试大纲（2018 年版）》为最新版本考纲，自2018年3月开始实施。

2019年全国计算机等级考试（NCRE）将举办四次考试，时间分别为3月30日至4月1日（第54次）、6月1日（第55次）、9月21日至23日（第56次）及12月7日（第57次）。其中3月和9月考试开考全部级别全部科目，6月和12月考试开考一、二级全部科目。各省级承办机构可根据实际情况决定是否开考6月和12月考试。
